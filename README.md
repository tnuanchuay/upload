# upload

if you want to upload file to your serve as static serve
this is it

### get
````
go get github.com/tspn/upload
cd $GOPATH/bin
cp upload /bin/
````

### usage
##### in your server
````
upload --serve
````
##### in your cliend
````
upload --host=ipaddr thefile
````
